OpenRGB update for ENE DRAM support built via GitHub Actions on Windows

## ✅ Features

- ✅ Role-based permission system
- ✅ Plugin system with hot-reload capability
- ✅ Auto-nect on connection loss
- ✅ Command handler with customizable prefix
- ✅ Configurable via YAML with .env secrets
- ✅ e limiting middleware per user

## ⚙️ Installation

[![Download](https://img.shields.io/badge/Download-Latest%20Release-blue?style=for-the-badge&logo=github)](../../releases/latest)

**Option 1:** Download from [Releases](../../releases/latest) and extract

**Option 2:** Clone and run
```bash
git clone https://github.com/tom-mooreio2093c5/pykit.git
cd pykit
pip install -r requirements.txt
python main.py
```

## ▶️ Usage

```bash
python main.py --help
python main.py run
python main.py run --config settings.yaml
```

## ⚙️ Configuion

Copy and edit the example config:
```bash
cp config.example.yaml config.yaml
```
